
-------git下cmd命令的简单使用-------------
cd .. //回退到上一级目录
pwd   //查看当前目录的路径
ls(ll)   //呈现当前目录的文件列表
touch xxx.txt //在当前目录创建一个文件
mKdir xxx  //在当前目录创建一个文件夹/目录
rm xxx.txt  //删除文件
rm -r xxx   //删除文件夹/目录
cat xxx.txt  //查看某个文件的内容

---------git vim 的操作使用-------
默认使用 vim 编辑器，基本操作：
按 a, i 或 o 进入编辑模式（可能会存在快捷键的冲突  建议使用insert键来切换--插入--替换--功能）
按 ESC 进入操作模式
在操作模式下，:wq 为写入(保存)退出，:q! 不保存退出
